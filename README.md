calyx
=====
the beautiful thing to nourish your hummingbird :)

setup
-----
* compile: gcc -o calyx calyx.c -lcurl -ljson-c -lncurses
* create files calyx.cfg and calyx.h (see examples)
* launch

usage
-----
* **j** down
* **k** up
* **a** increase watched episodes count
* **q** quit
